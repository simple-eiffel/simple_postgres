Overview

simple_postgres provides a facade for PostgreSQL database operations. It wraps libpq with Eiffel idioms, offering connection management, query execution, result set access, NULL handling, and transaction support.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries with full Design by Contract support.

Quick Start

Installation

Set environment variables and add to your ECF:

-- Set POSTGRESQL_HOME and PGPASSWORD
-- Set SIMPLE_EIFFEL=D:\prod
<library name="simple_postgres" location="$SIMPLE_EIFFEL/simple_postgres/simple_postgres.ecf"/>

Basic Query

local
    db: SIMPLE_POSTGRES
do
    create db.make
    db.connect ("host=localhost dbname=mydb user=postgres")

    if db.is_connected then
        db.query ("SELECT name, email FROM users")
        if attached db.last_result as res then
            across 0 |..| (res.row_count - 1) as row loop
                print (res.value_at (row, 0) + ": " + res.value_at (row, 1))
            end
            res.clear
        end
        db.disconnect
    end
end

Transaction Example

db.execute ("BEGIN")
db.execute ("UPDATE accounts SET balance = balance - 100 WHERE id = 1")
db.execute ("UPDATE accounts SET balance = balance + 100 WHERE id = 2")
db.execute ("COMMIT")

Features

Connection Management

Connect and disconnect with standard PostgreSQL connection strings.

Query Execution

Execute SELECT queries and access result sets with row/column indexing.

DML Support

INSERT, UPDATE, DELETE with affected row counts.

Transaction Support

BEGIN, COMMIT, ROLLBACK for atomic operations.

NULL Handling

Proper detection and handling of NULL values in results.

Design by Contract

Full preconditions, postconditions, and class invariants.

API Reference

SIMPLE_POSTGRES

FeatureDescription
makeCreate disconnected client
connect (conn_string)Connect to database
disconnectClose connection
is_connectedCheck connection status
query (sql)Execute SELECT, store result
execute (sql)Execute INSERT/UPDATE/DELETE
last_resultResult from last query
last_errorError message (empty if none)
affected_rowsRows affected by last execute

PG_RESULT

FeatureDescription
row_countNumber of rows
column_countNumber of columns
value_at (row, col)Get value (Void if NULL)
column_name (col)Get column name
is_null_at (row, col)Check if value is NULL
is_emptyCheck if no rows
clearFree result memory
